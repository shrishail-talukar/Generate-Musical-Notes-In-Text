# Generate musical notes, and display in random fashion

**Usage:**

`./create_notes.sh [srgmpdnS or anyset of notes] [range_of_notes_in_int] [delay_before_next_note_in_s]`

**Pre-requisite:**
For Ubuntu:

`apt install crunch`
